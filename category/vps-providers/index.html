      VPS商家选择指南 #

      🔒 我们的承诺 #

      🌟 中立评测原则 #

      1. 零付费推广
        所有测评商家均不接受赞助或佣金，测试用服务器全部自费购买
      2. 长期追踪机制
        • 持续监控商家关键指标（在线率/工单响应速度）
        • 季度性复测硬件性能
      3. 用户反馈验证
        每篇测评参考至少30位真实用户评价，标注争议事件（如2025年商家C大规模宕机）

      🔍 选购商家必查清单 #

      在购买VPS前，请务必审查以下关键信息，确保商家符合您的需求，避免踩坑：

      1. 退款政策（Money-Back Guarantee） #

      ✅ 理想标准：

      • 无条件退款期（至少3-7天，部分优质商家提供30天）
      • 按小时/按天计费（适合短期测试）
      • 退款方式（原路退回，而非仅限余额）

      ❌ 风险提示：

      • “试用期” ≠ 退款期（部分商家试用期后不退）
      • 隐藏条款（如仅限新用户首单可退）

      2. 服务条款（TOS/AUP） #

      ✅ 必查内容：

      • CPU/带宽限制（是否标明具体阈值？）
      • 数据备份责任（商家是否提供免费快照？）
      • IP更换政策（被封IP后是否收费更换？）

      ❌ 危险信号：

      • “无限流量但限制速度”（实际可能限速1Mbps）
      • “有权随时终止服务”（无明确违规标准）

      3. 支付与账户安全 #

      ✅ 推荐选择：

      • PayPal/信用卡支付（支持争议申诉）
      • 两步验证（2FA）支持（防盗号）

      ❌ 谨慎选择：

      • 仅支持加密货币（争议难解决）
      • 要求身份证验证（隐私风险）

      📌 终极自查：
      ▢ 我已阅读退款政策
      ▢ 我确认商家无隐藏限制（CPU/带宽）
      ▢ 我选择的支付方式可争议退款
      ▢ 我检查过近期用户评价

      ⚠️ 注意：即使是大牌商家，也可能有特殊限制（如AWS轻量套餐禁止代理），务必细读条款！
