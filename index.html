        🧭 新手选购指南 #

        1. 🌐 网络与连接性 #

        核心关注点：

        • 线路质量：
          • 优选：CN2 GIA/GT等针对中国优化的线路
          • 避免：普通国际线路（晚高峰可能拥堵）
        • 机房位置：
          • 优选：距离近的数据中心机房
          • 避免：偏远地区的服务器机房
        • 带宽流量：
          • 基础需求：1Gbps带宽 + 1TB/月流量
          • 大流量需求：选择不限流量（注意限速条款）

        2. 💻 硬件配置解析 #

        性能阶梯：

        组件优选方案平替方案最次选项
        CPUAMD EPYC/Intel Xeon普通虚拟化CPU不明型号
        内存DDR4/DDR5DDR3共享内存
        存储NVMe SSDSATA SSDHDD机械盘
        IP独立IPv4共享IPv4纯IPv6

        配置参考：

        • 个人博客：1核1G+NVMe 20GB
        • 外贸网站：2核4G+SSD 50GB
        • 数据备份：大容量HDD存储

        3. ⚠️ 必查细节 #

        1. 网络测试：
          • 索取测试IP（用ping.pe测路由）
          • 晚高峰速度测试（20:00-23:00）
        2. 条款确认：
          • 是否允许重装系统
          • 流量超额处理方式
          • 是否提供root权限
        3. 支付与售后：
          • 支持支付宝/微信支付
          • 工单响应时间
          • 退款政策（至少3天无理由）

        4. 🔍 验证工具推荐 #

        • 网络测试：ping.pe / ipip.net
        • 性能测试：bench.sh / unixbench
        • 真伪验证：whois查商家运营年限
